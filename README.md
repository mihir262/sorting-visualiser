# Sorting Visualiser

A simple C + raylib app that visualizes sorting bars and swap operations.

## Build and run

```bash
make
make run
```

## Controls

- `1` Bubble sort
- `2` Insertion sort
- `3` Selection sort
- `4` Quick sort
- `SPACE` Start sorting animation
- `R` Reshuffle bars

## Notes

- Rendering runs at 60 FPS.
- Swap animation speed is controlled by `swapDelaySeconds` in `main.c`.
